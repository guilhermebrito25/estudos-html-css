Parágrafos e quebras de linha

Você pode escrever um parágrafo de quaalquer jeito, basta apenas por o texto no meio das tags de parágrafo.

Se precisar quebrar o texto em qualquer lugar especifico é so usar a tag (br).
Exemplo:
Lorem ipsum dolor sit amet consectetur adipisicing elit.
Esse tenetur deleniti deserunt modi ducimus voluptates commodi laborum, molestias non voluptatibus, facilis consequatur incidunt! Laborum, minima minus? Voluptates, vero. Maiores, exercitationem.
